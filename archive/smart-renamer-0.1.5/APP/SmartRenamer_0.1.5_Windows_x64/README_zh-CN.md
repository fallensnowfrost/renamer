# SmartRenamer 0.1.5发布包

这是SmartRenamer的Windows x64发布说明目录。主程序exe请从GitHub Releases下载。

## 运行程序

下载Release资产后双击运行：

```text
SmartRenamer_0.1.5_Windows_x64_Portable.exe
```

## 包内文件

- `SmartRenamer_0.1.5_Windows_x64_Portable.exe`：主程序，作为GitHub Release资产提供。
- `docs/UserGuide_zh-CN.md`：用户手册。
- `docs/ReleaseNotes_0.1.5_zh-CN.md`：版本发布说明。
- `docs/FileManifest_0.1.5.md`：发布文件清单。
- `docs/Checksums_SHA256.md`：SHA256校验信息。
