# SmartRenamer 0.1.5发布说明

发布日期：2026-06-10  
平台：Windows x64  
包类型：Portable exe

## 本版重点

- 修正文件重命名规则：文件名只生成纯序号，不再包含人物名称或主题。
- 图片和视频在同一文件夹内分别独立编号。
- 扫描不再强制要求输入人物名称。
- 新增执行进度显示。
- 操作按钮调整为底部固定栏，减少滚动操作。
- 冲突项改为跳过，不再阻止整批后续任务执行。
- 撤销逻辑只回滚实际执行成功的项目。
- 更新个性化应用图标。
- 文件夹名称清理改为独立操作卡片，每类清理都有自己的预览和执行按钮。
- 新增智能规则删除、高级正则删除和半自动无用内容推荐。
- 新增对`DJAWA Photo Vol 0216`、开头编号`048`等疑似无用片段的推荐清理。

## 功能范围

本程序支持：

- 子文件夹递归扫描。
- 子文件夹重命名。
- 媒体文件纯序号重命名。
- 文件夹名称文本清理。
- 正则表达式删除和不会正则时可用的智能规则删除。
- 首尾空格清理。
- 执行前预览。
- 冲突和警告提示。
- 最近一次操作撤销。
- 导出操作日志。

本程序不支持：

- 批量删除文件。
- 批量删除目录。
- 自动上传或联网处理媒体文件。

## 命名规则

文件夹命名：

```text
[人物名称] - 主题 [图片数量P 视频数量V 总大小]
```

文件命名：

```text
001.jpg
002.jpg
001.mp4
002.mp4
```

编号位数由当前文件夹内同类型媒体数量决定。

## 已知说明

- 本版本为本地离线工具，直接运行exe即可使用。
- 由于是本地打包程序，部分Windows环境可能出现安全提醒，请确认来源后再运行。
- 若文件被其他程序占用，重命名可能失败，关闭占用程序后重新扫描即可。
