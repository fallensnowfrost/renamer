# werenameit 0.2.0-alpha.1本地包说明

这是werenameit的未发布版本本地包目录。

## 本地打开

双击下面这个文件即可打开程序：

```text
APP\werenameit_0.2.0-alpha.1_Windows_x64\werenameit.exe
```

外层`werenameit.exe`是小启动器，真实Electron运行文件放在同级`app\`目录中。使用时请保持`werenameit.exe`和`app\`在同一个文件夹内。
