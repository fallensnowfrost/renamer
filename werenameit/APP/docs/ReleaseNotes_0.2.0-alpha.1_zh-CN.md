# werenameit 0.2.0-alpha.1发布说明

状态：未发布版本 / Unreleased

## 本次变化

- 新建`werenameit`新版项目目录。
- 程序名称统一为`werenameit`。
- 版本号设置为`0.2.0-alpha.1`。
- 页面布局改为固定顶部控制区、固定操作区、独立滚动文件夹预览、右侧详情和清理工具。
- 生成文件夹式本地包：
  - 外层小启动器：`werenameit.exe`
  - 真实Electron运行目录：`app/`
- 旧版`SmartRenamer 0.1.5`资料保留在`archive/smart-renamer-0.1.5`。

## 注意事项

- 这是未发布版本，不覆盖GitHub Release`v0.1.5`。
- 使用时请保持外层`werenameit.exe`和`app/`目录在同一个文件夹内。
- 不提供任何批量删除文件或目录功能。
